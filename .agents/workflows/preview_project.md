# Preview Project

To run a local web server and preview this project on your computer:

1. Open a terminal in the root folder (`c:\HYUNGJOO\cat_website`).
2. Run the following command:
   ```bash
   python -m http.server 8000
   ```
   *Note: If `python` is not recognized, you can also use `npx serve`.*

3. Once the server is running, the terminal will show that it is serving on port 8000. 
4. Click the link `http://localhost:8000` or use the Antigravity local port forwarding preview button to view your site!
